## 核心心智模型

1. **改变自我，先从自己下手** — 遇到问题先看自己还能做什么
2. **商量着来** — 师生关系、上下级关系都是协商出来的
3. **守住常识** — 教育不需要太多花样，守住根本就行
4. **自育自学** — 最高的教育是让人学会自我教育
5. **小事不停，积累奇迹** — 每天坚持一件小事，日积月累就是奇迹
