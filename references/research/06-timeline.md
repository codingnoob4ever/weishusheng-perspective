## 生平概览

**魏书生**（1950年5月4日—），出生于河北省交河县（今泊头市），中国当代著名教育改革家。

---

## 详细年表

### 早年时期（1950-1968）

| 年份 | 事件 |
|------|------|
| 1950年 | 5月4日出生于河北省交河县 |
| 1956年 | 随父母迁居沈阳 |
| 1957年 | 就读于沈阳和平区砂山一校 |
| 1963年 | 小学毕业，考入沈阳市三十二中学读初中 |
| 1965年 | 接触对其此后生活发生重大影响的两本哲学书籍：《辩证唯物主义纲要》《哲学讲义》——**思想启蒙起点** |
| 1966年 | 初中毕业，因"文革"开始，失去升入高中的机会 |
| 1968年 | 到辽宁省盘锦地区新建农场当下乡知识青年 |

### 教师生涯起步（1969-1978）

| 年份 | 事件 |
|------|------|
| 1969年 | 受上级指派在新建农场的红旗小学教书（19岁当民办教师） |
| 1971年 | 被招工到盘锦电机厂工作，任政工干事（21岁进工厂） |
| 1972年 | 撰写专著《谈改造世界观》（未出版） |
| 1973年 | 撰写专著《谈工作方法》（未出版），被评为厂级领导接班人 |
| 1974年 | 加入中国共产党；**因其尊孔及否定文化大革命受到批判，被工厂停职审查达一年之久** |
| 1978年 | 2月，到盘锦盘山县三中做语文教师（28岁正式走上讲台）；9月，被任命为教导处主任 |

> **关键背景**：魏书生在电机厂工作期间，向各级领导提出当教师申请达150多次，历时6年才如愿。这种执着被他自己称为"六载夙愿"。

### 教改探索期（1979-1986）

| 年份 | 事件 |
|------|------|
| 1979年 | 3月开始第一轮教改实验；12月在营口地区语文教研会上宣读论文《作文教学改革》反映强烈 |
| 1980年 | 撰写《思想政治教育与开发智力资源》发表于《辽宁教育》杂志 |
| 1981年 | 2月，被辽宁省政府授予"优秀班主任"称号；开始为《文科教学参考》杂志写作专栏文章 |
| 1982年 | 5月写作论文《初中"推普"五题》；6月被中共辽宁省委和省政府授予"劳动模范"称号；9月，中共盘山县委和县政府联合发出《关于在全县教育系统开展向魏书生同志学习的决定》 |
| 1983年 | 继续教改探索，影响力扩大 |
| 1984年 | 被授予"特级教师"称号；被评为全国优秀班主任；**语言学家吕叔湘评价："魏书生是个教育家，而且不是一般的教育家"** |
| 1986年 | 3月被任命为盘锦市实验中学校长兼书记 |

### 荣誉巅峰期（1987-1997）

| 年份 | 事件 |
|------|------|
| 1987年 | 被评为省特等劳动模范 |
| 1988年 | 被评为全国劳动模范 |
| 1989年 | 被评为全国中青年有突出贡献的专家 |
| 1990年 | 获首届"中国十大杰出青年"称号 |
| 1995年 | 兼任盘锦市教委副主任 |
| 1997年 | 10月被任命为盘锦市教育局局长兼党委书记；**同时继续兼任一个班的语文课** |

> 这一时期魏书生成为中共十三大、十四大、十五大、十六大、十七大代表（连续五届），达到政治荣誉巅峰。

### 行政管理者时期（1998-2010）

| 年份 | 事件 |
|------|------|
| 1998-2010 | 任盘锦市教育局局长兼党委书记，同时坚持每周为学生上一节语文课 |
| 2002年 | 被天津市教育科学研究院聘为研究生导师；被国家教育行政学院聘为兼职教授 |
| 2010年 | 9月卸任盘锦市教育局局长、党组书记 |

## 思想转折点

1. **1965年**：接触哲学书籍，形成辩证思维底色
2. **1974年**：因"尊孔"和"否定文革"受批判——早期思想独立的证明
3. **1978年**：终于走上教师岗位——人生理想实现的关键节点
4. **1984年**：吕叔湘评价其为"教育家"——外部权威认可
5. **1997年**：任教育局长但仍坚持上课——实践者的身份自觉
6. **2024年**：出现系统性批评——社会评价开始分化
